# 更新日志

### 本分支特性

本分支由 Jingshiro 维护，项目地址：[https://github.com/Jingshiro/legado](https://github.com/Jingshiro/legado)

- **详细阅读记录**：记录更细致的阅读信息，支持导出与统计。
- **AI 助手**：引入大模型交互能力，支持工具调用，AI 可查询和管理书架、书源、订阅等数据。支持 AI Agent 通过 HTTP 调用，可以将此skills发送给你的 AI Agent 使用：`https://github.com/Jingshiro/reading-skill`
- **想法分享与导出**：长按文字可写下想法，可生成分享卡片，也可一键导出至 Obsidian。
- **阅读小票**：可在书籍首页和尾页显示小票，录入书籍的评分和阅读时长。
- **WebDAV 备份管理增强**：支持在本地对云端备份进行删除、重命名等操作。
- **主题全量导出功能**：支持一键分享全部主题设置。
- **读完/刷书标签**：记录书籍阅读状态（读完/N刷），并在书架上进行展示。

**2026/07 - 体验优化**

- 新增高亮规则功能：替换规则新增"高亮模式"，支持对正则捕获组应用 HTML 样式（加粗、斜体、下划线、颜色、字号等），工具栏可一键插入标签，支持自定义颜色输入
- Web API 安全警告：启动 Web 服务时弹出安全警告提示
- Obsidian 静默自动导出：新增自动导出开关，开启后新增、修改或删除想法时自动同步到 Obsidian，无需手动操作
- 阅读记录排除订阅源：RSS 订阅源的阅读行为（文章阅读、视频/音频播放）不再记录到阅读历史，数据更精准
- 高亮规则性能优化：引入 LRU 缓存机制，避免对相同内容重复执行正则匹配和样式处理，翻页更流畅


**2026/06 - 阅读页体验优化**

- 主题美化：新增底栏图标自定义功能，可为书架、发现、订阅、我的分别选择自定义图标；启动页支持GIF格式；优化阅读页二级弹窗样式；新增想法下划线样式自定义功能；阅读小票样式优化；优化想法卡片设计；修复启动页自定义背景图和文字/图标开关不生效的问题；优化"我的"页面布局，主题模式归入设置分区，书签改为书签与想法并整合导出到Obsidian功能，备份与恢复移至其它分区

- AI助手优化：发送章节内容时会附上用户想法
