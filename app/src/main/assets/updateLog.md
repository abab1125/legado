# 更新日志

### 本分支特性

本分支由 abab1125 维护，基于 Jingshiro/legado 改造，专注互动写作能力。

- **写作台 V1**：章纲编辑（AI 生成/手动）、提示词管理、知识点管理，全部集成到底部 Tab
- **目录增强**：每章节 ⋮ 菜单支持插入/起名/移动/删除/整理序号
- **提示词全局化**：不再绑定书籍，所有书的提示词互通
- **知识点全局化**：移除 bookUrl，全局共享知识库

**2026/06/27 — 写作台重构**

- 目录页每章节新增 ⋮ 菜单，支持 8 项操作：插入下一章、自动起名、手动起名、批量章节起名、整理章节序号、上移、下移、删除
- 底部 Tab 新增「提示词」「知识点」两个入口（与书架/发现同级）
- 提示词管理页：增删改，分类（角色设定/世界观/写作风格/大纲）
- 知识点管理页：增删改，分类（人物/地点/事件/笔记/其他）
- 知识点数据库重构：去掉 bookUrl，改为全局共享，migration 98→99
- 旧写作台 Dialog（PromptManageDialog/KnowledgeManageDialog）保留兼容

**2026/07/09 — 提取角色 + 三级钻取知识库**

- AI 聊天新增「提取角色」功能，多选章节调用 LLM 批量提取小说角色卡
- 知识库管理页三级钻取：分类 → 小说名分组 → 角色卡
- 角色卡支持 @引用，引用标题带「小说名/」前缀

**2026/07 - 体验优化（上游 Jingshiro）**

- 新增高亮规则功能：替换规则新增"高亮模式"，支持对正则捕获组应用 HTML 样式（加粗、斜体、下划线、颜色、字号等），工具栏可一键插入标签，支持自定义颜色输入
- Web API 安全警告：启动 Web 服务时弹出安全警告提示
- Obsidian 静默自动导出：新增自动导出开关，开启后新增、修改或删除想法时自动同步到 Obsidian，无需手动操作
- 阅读记录排除订阅源：RSS 订阅源的阅读行为（文章阅读、视频/音频播放）不再记录到阅读历史，数据更精准
- 高亮规则性能优化：引入 LRU 缓存机制，避免对相同内容重复执行正则匹配和样式处理，翻页更流畅
- 新增 AI 助手与 Web API 接口支持：AI 助手新增高亮净化规则和主题配色的查询与修改工具，Web API 同步开放对应接口
