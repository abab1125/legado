# 替换为 语法说明

## 什么是捕获组？

在正则表达式中，用小括号 `()` 括起来的部分叫做**捕获组**。

例如正则：`你好(.*?)再见`

匹配文本 `你好世界再见` 时：
- `$0` = 整个匹配 → `你好世界再见`
- `$1` = 第1个括号里的内容 → `世界`

多个括号的情况：
- 正则：`(第.*?)章(.*?)内容`
- `$0` = 整个匹配
- `$1` = 第1个括号的内容
- `$2` = 第2个括号的内容

---

## 支持的 HTML 标签

在高亮模式下，可以用以下标签设置样式：

| 标签 | 效果 | 示例 |
|------|------|------|
| `<b>...</b>` | **加粗** | `<b>$1</b>` |
| `<i>...</i>` | *斜体* | `<i>$1</i>` |
| `<u>...</u>` | 下划线 | `<u>$1</u>` |
| `<font color="red">...</font>` | 颜色 | `<font color="red">$1</font>` |
| `<font face="楷体">...</font>` | 字体 | `<font face="楷体">$1</font>` |
| `<span style="font-size:18px">...</span>` | 字号 | `<span style="font-size:18px">$1</span>` |
| `<big>...</big>` | 大号字 | `<big>$1</big>` |
| `<small>...</small>` | 小号字 | `<small>$1</small>` |
| `<div style="text-align: center;">...</div>` | 居中 | `<div style="text-align: center;">$1</div>` |

支持的颜色名：red, blue, green, orange, purple, brown, pink, gold, gray, black

---

## 常用写法示例

### 把匹配的文字变红色
```
替换为：<font color="red">$0</font>
```

### 把匹配的文字加粗
```
替换为：<b>$0</b>
```

### 加粗 + 红色（标签可以嵌套）
```
替换为：<b><font color="red">$0</font></b>
```

### 只对第1个捕获组加粗，第2个变红色
```
正则："(.*?)"和'(.*?)'
替换为：<b>$1</b>和<font color="red">$2</font>
```

### 居中显示
```
替换为：<div style="text-align: center;">$0</div>
```
> 适合配合"作用于标题"使用，让章节标题居中显示。

### 更换字体
```
替换为：<font face="楷体">$0</font>
```
> 适合全文匹配 `^[\s\S]*$` 时统一更换正文字体。

### 丢弃某个捕获组（不显示）
```
正则：第(.*?)章(.*?)
替换为：第$1章
（$2 不写就不显示）
```

---

## 注意事项

1. `$0` = 整个匹配到的文本（包含匹配到的所有内容）
2. `$1` = 第1个括号捕获的内容，`$2` = 第2个，以此类推
3. 标签可以嵌套，如 `<b><font color="red">$1</font></b>`
4. 高亮模式需要开启 **使用正则表达式**
